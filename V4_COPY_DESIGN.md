# v4 文言設計・承認用サンプル

> 状態: 方針確認用。以下の改善案は、承認されるまで実装データへ反映しない。
>
> v4は文言のみの検討版とし、v3の表示・配置・モーションは変更しない。全54手の現行表示は `V4_WORDING.md` に保存する。

## コンセプト

**一手を覚えるだけでなく、動作・理由・心・歴史がつながる稽古プレイヤー。**

画面では、まず正確な動作が迷わず読めることを優先する。そのうえで、所作を見る目、客を思う心、道具や茶の湯の背景を、それぞれ重複しない層で伝える。

### Aboutページへ残す要点

- 点前を暗記する教材ではなく、一手の意味まで理解する入口にする。
- 「何をするか」「どう整えるか」「なぜそうするか」「どこから来た文化か」をつなぐ。
- 表千家・風炉・薄茶・運び点前・女点前という前提を明示する。
- 流儀差、男女差、社中差、要監修箇所は曖昧に混ぜず、適用条件を表示する。
- 文言を正本とし、承認後の動きは文言に合わせて設計する。
- 道具を美しく見せるだけでなく、道具を扱う人と客との関係を伝える。

## 情報の役割

| 表示 | 答える問い | 書く内容 | 書かない内容 |
|---|---|---|---|
| タイトル | 今、何をする？ | 一手の中心動作 | 抽象的な情緒 |
| 本文 | 何を、どの順で？ | 観察できる動作、手、方向、回数、置き先 | 心構え、歴史 |
| 所作の要点 | どこを見ればよい？ | その一手固有の確認点 | 本文の言い換え |
| 一手の心 | なぜ大切？ | 客への配慮、動作の意味 | 手順の再説明 |
| 女点前の違い | 何が異なる？ | 男点前との差がある場合だけ | 男女差でない注意 |
| 取扱注意 | 何を傷めやすい？ | 熱、傷、落下、音への注意 | 女点前欄への混在 |
| 用語 | この言葉は何？ | 読みと一行定義 | 長い歴史解説 |
| 豆知識 | 背景に何がある？ | 道具、人物、産地、季節、歴史 | 手順、心得の反復 |
| 動作ピクト | 何がどう動く？ | 対象＋役割＋動詞 | 道具名だけの羅列 |

豆知識は全手へ強制的に割り当てない。直接関係する話題がある手だけに表示し、一般的な歴史や茶の湯の心はAboutまたは章の導入へまとめる。

## 動作ピクトの設計

本番では同じ線幅・同じ視覚重量の単色SVGを使う。下記の記号はMD上の意味確認用であり、完成アイコンではない。

| 系統 | 仮記号 | 用途 |
|---|---:|---|
| 手・保持 | `H` | 右手、左手、両手、持ち替え |
| 移動 | `M` | 運ぶ、回す、上げる、下げる |
| 位置 | `P` | 据える、置く、戻す、注ぎ先 |
| 道具 | `T` | その手で実際に扱う道具 |
| 湯水 | `W` | 湯、水、汲む、注ぐ、あける |
| 発話 | `S` | 正客・亭主の言葉、応答順 |
| 確認 | `C` | 穂先、正面、位置、終了状態 |

表示は最大3個を基本とし、`アイコン｜対象｜動作または役割` の順に読む。例えば「茶碗／茶筅」という名詞列ではなく、`T｜茶筅｜穂先を確かめる` とする。背景にあるだけの道具は表示しない。

## サンプル1 — 第5手

- **フェーズ:** 清め / 据える
- **タイトル:** 蓋置を据え、柄杓を構える
- **本文:** 左手で柄杓を少し上げ、右手で建水から蓋置を取り出す。柄杓をいったん建水へ戻し、蓋置を風炉の左へ据える。柄杓を持ち直し、蓋置へ引いて構える。
- **所作の要点:** 柄杓を上げる、蓋置を出す、柄杓を戻す、蓋置を据える、柄杓を構える、の順を崩さない。
- **一手の心:** 二つの道具を同時に急いで扱わず、一手ずつ置きどころを定めることで、点前の場を整える。
- **女点前の違い:** 左手で柄杓を上げ、右手で蓋置を出す。男点前とは扱いが異なる。
- **取扱注意:** なし
- **用語:** `蓋置（ふたおき）｜釜の蓋や柄杓を受ける道具`
- **動作ピクト:** `H｜左手｜柄杓を上げる` / `H｜右手｜蓋置を出す` / `P｜風炉の左｜据える`
- **豆知識:** 表示候補なし。蓋置の機能は用語欄で説明できるため、重ねない。

- **取扱注意:** なし

- **動作ピクト:** `M｜茶碗｜反時計回りに二度` / `C｜正面｜客へ向ける` / `P｜定座｜出す`
- **豆知識:** `茶碗の正面と、主客の向き` — 亭主が正面を客へ向ける意味と、客の扱いへつながる関係を紹介する。

## サンプル5 — 第2手

- **フェーズ:** 清め / 運び出し
- **タイトル:** 水指を運び、定座へ据える

- **女点前の違い:** なし
- **取扱注意:** 水指を傾けず、置く直前まで両手で安定させる。
- **用語:** `水指（みずさし）｜点前で使う水を蓄えておく器` / `定座（じょうざ）｜道具を置く定められた位置`
- **動作ピクト:** `H｜水指｜両手で持つ` / `M｜足運び｜左足から六歩` / `P｜風炉の右横｜据える`
- **豆知識:** `運び点前は基本の形` — 棚を使わず、道具を席へ運び出して行う点前であることを紹介する。

## 全54手へ反映するときのルール

1. 先に `V4_WORDING.md` 上で全項目を承認し、その後データへ移す。
2. 本文は動作だけにし、気持ちや歴史を混ぜない。
3. 所作の要点は各手につき一つを基本とし、見て確認できる言葉で書く。
4. 「静かに」「丁寧に」「心を整える」を定型句として反復しない。
5. 女点前欄は男女差がある手だけに表示する。
6. 豆知識は関連性が高い手だけに置き、同じ話題を連続表示しない。
7. 動作ピクトは最大3個を基本とし、道具名ではなく役割と動詞を示す。
8. 専門語は消さず、初出時に読みと短い定義を添える。
9. 社中差や未確認事項は断定せず、監修状態を記録する。
10. 文言承認後、各手の開始状態・動作・終了状態に合わせてモーションを設計する。
