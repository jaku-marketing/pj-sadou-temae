# 表千家 お点前シミュレーション

表千家・風炉（夏）・薄茶・運び点前・**女点前**を、自分目線で一手ずつ追えるWebプレイヤーUIです。

## 内容
- `player.html` … 没入フルスクリーン型プレイヤー（Three.js 季節アニメ・縦書き手番・フィルムストリップ）
- `data/steps.js` … お点前データ層（img3〜53／全51手）。描画と分離。
- `data/tidbits.js` … 茶道の豆知識（千利休・歴史・季節・道具・作法など）
- `img/` … 各手の実シーン画像（1〜2＝表紙、3〜53＝手順）
- `PROCEDURE.md` … 正式手順の一次資料まとめ
- `UI_DESIGN.md` … インターフェース設計書（白×ピンク・明朝・縦書きルール）

## 使い方
`player.html` をブラウザで開く（ビルド不要）。

## ロードマップ
- UI ブラッシュアップ（クックモード型＋進捗パス、説明の表示/非表示トグル）
- 将来：Three.js / 3D（Fable 5）でのパーツ化アニメーション
- インデックスの分岐（流儀 × 季節 × 性別）

表千家／風炉（夏）／女点前／薄茶・運び を起点に拡張予定。
